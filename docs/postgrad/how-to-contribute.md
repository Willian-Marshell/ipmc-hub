# 如何贡献升学就业经验

这个板块沉淀**走过的路**，让后来者少绕弯。

## 推荐分区

```
docs/postgrad/
├── _category_.json
├── overview.md            方向地图：保研 vs 考研 vs 留学 vs 就业
├── grad-domestic/         保研与考研
├── grad-overseas/         留学申请
├── jobs/                  就业（实习、秋招、行业方向）
└── timelines/             各方向时间线模板
```

## 至少要回答的几个问题

1. 这条路的**关键节点**与**时间线**？
2. **硬指标**门槛（GPA / 英语 / 科研 / 竞赛）大概是什么？
3. **决策依据**：为什么选这条路而不是其他？
4. 对学弟学妹的**一条具体建议**？

## 隐私边界（重要）

升学就业经验天然涉及个人信息，请遵守：

- **不写**他人的录取或拒信结果（除非本人同意公开）
- **不写**导师或 HR 的负面评价（写「沟通方式偏正式」而不是「很难搞」）
- **可写**自己的 GPA、英语成绩、科研经历，但已被录取的请脱敏到区间（如「前 10%」而非具体名次）
- **不上传**任何包含他人信息的截图（聊天记录、邮件原文）

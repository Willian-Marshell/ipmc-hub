# 欢迎来到 IPMC Hub

这里是深圳大学「智能感知与运动控制（IPMC）」本科微专业学生共同维护的开源知识库。

我们相信：每一届踩过的坑、攒下的经验、做过的项目，都不该在毕业那天随简历一起消失。把它们写下来、存起来、传下去，下一届就能站在更高的起点上出发。

## 这里有什么

- **[课程笔记](./category/课程笔记)** — 微专业各门课的通关攻略、实验避坑、推荐资源
- **[比赛经验](./category/比赛经验)** — RoboMaster、电赛、数模等赛事的技术栈与历届复盘
- **[AI 工具链](./category/ai-工具链)** — Agent 使用心得、Prompt 库、本地部署 SOP
- **[升学就业](./category/升学就业)** — 保研、考研、留学、就业方向的经验合集
- **[心得博客](/blog)** — 个人视角的学期复盘、比赛日记、面试回忆

## 我想读资料

直接逛上面任一板块即可。读到错别字或过时内容，每页底部都有「Edit this page」可以一键跳到 GitHub 提 PR。

## 我想贡献

万事开头难——但你提的第一个 PR 永远是最有价值的那个。

新贡献者从 [10 分钟提出第一个 PR](./contributing/quickstart) 开始，全流程不超过半小时。

## 为什么做这个

- 让微专业学生的学习过程少一点迷茫
- 让贡献被看见、被记住、被奖励（见[开源之星计划](./contributing/star-program)）
- 让非微专业的同学也能受益，吸引更多人加入

宁缺毋滥，但人人可参与。

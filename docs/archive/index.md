# 关于本归档区

这里存放被判定为过时（`status: archived`）的内容。

按 [退役策略](../contributing/archival-policy)，IPMC Hub 的内容**只搬不删**：
- 即便信息已经不再适用今天的课程或赛规，原文依然有历史参考价值
- 学弟学妹可以从这里看到三五年前学长学姐的视角，理解知识如何演化

如果你发现归档内容里有需要更正或补充的地方，欢迎在新文档中引用并写一篇「续篇」。
